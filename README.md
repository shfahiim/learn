# 41

## Current Modules

- **Probability**: From foundations like Bayes' Theorem to advanced topics like the Central Limit Theorem and Students-T distribution.
- **Machine Learning**: A comprehensive guide covering Supervised/Unsupervised learning, Linear Models, Kernel Methods, and Sequential Data.

## Features

- **Interactive Visualizations**: Real-time charts and simulations powered by Recharts.
- **MDX Powered**: High-quality mathematical notation using LaTeX.
- **Clean UI**: Minimalist and accessible design focused on reading and learning.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Production Build

To create a production-ready build:

```bash
npm run build
```
