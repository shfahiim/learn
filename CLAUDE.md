# ML Lecture Notes Processing Instructions
When the user dumps ML lecture notes:
1. **Extraction**: Extract every detail, concept, mathematical notation, theory, and math.
2. **Translation**: Translate all notes written in Bangla or a mix of Bangla and English into clear, academic English.
3. **Visuals**: Carefully analyze and interpret any tables, graphs, and hand-drawn diagrams (including pen drawings). Do not miss any of them. Describe them accurately in text or recreate them.
4. **Augmentation**: Add extra supporting diagrams or explanatory visuals if it helps clarify the concepts, but ensure original diagrams are fully captured.
5. **Integration Planning**: After analyzing the content, plan where to appropriately integrate these new concepts, math, and diagrams into the existing course/notes directory structure.
