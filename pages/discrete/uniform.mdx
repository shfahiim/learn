# Uniform Distribution (Discrete)

<Callout type="info" emoji="🟰">
The Discrete Uniform Distribution describes a scenario where there is a finite, distinct set of possible outcomes, and **every single one of those outcomes is equally likely to occur.**
</Callout>

## Core Concepts

If a random variable $X$ follows a Discrete Uniform Distribution between a minimum integer $a$ and a maximum integer $b$, we write:
$$
X \sim \mathcal{U}\{a, b\}
$$

Where:
*   <span style={{color: '#8b5cf6', fontWeight: 'bold'}}>$a$</span> is the minimum possible integer value.
*   <span style={{color: '#ec4899', fontWeight: 'bold'}}>$b$</span> is the maximum possible integer value.
*   <span style={{color: '#3b82f6', fontWeight: 'bold'}}>$n = b - a + 1$</span> is the total number of possible outcomes.

### Probability Mass Function (PMF)

Because every outcome is equally likely, the probability of any specific outcome $x$ is simply $1$ divided by the total number of possible outcomes ($n$).

$$
P(X = x) = \begin{cases} 
\frac{1}{n} & \text{for } x = a, a+1, \dots, b \\
0 & \text{otherwise}
\end{cases}
$$

### Cumulative Distribution Function (CDF)

The probability that the outcome is less than or equal to $x$ grows linearly step-by-step:

$$
F(x) = \frac{x - a + 1}{n} \quad \text{for } a \le x \le b
$$

## Key Metrics

*   **Expected Value (Mean):** $E[X] = \frac{a + b}{2}$ *(Exactly in the middle)*
*   **Variance:** $\text{Var}(X) = \frac{n^2 - 1}{12}$

## Real-World Examples
1.  **Rolling a Fair Die:** A standard 6-sided die follows $\mathcal{U}\{1, 6\}$. The probability of rolling any specific number is exactly $\frac{1}{6}$.
2.  **Drawing a Card:** Drawing a single card from a well-shuffled standard deck of 52 cards (ignoring suits).
3.  **Lottery Balls:** Selecting a numbered ping-pong ball from a rotating lottery drum.

<MathExample title="Example: Discrete Uniform Variance">
  <MathQuestion>
    A fair 6-sided die is rolled. Values $a=1$ and $b=6$. What is the Expected Value and Variance of the roll?
  </MathQuestion>
  <MathSolution>
    **Expected Value:** $\frac{a+b}{2} = \frac{1+6}{2} = 3.5$
    
    **Variance:** $\frac{(b-a+1)^2 - 1}{12}$
    $\frac{(6-1+1)^2 - 1}{12} = \frac{6^2 - 1}{12} = \frac{35}{12} \approx 2.917$
  </MathSolution>
</MathExample>
